TouchFishGames : 新Item的追加方法

Created by 杨志庄 on 12月 20, 2022

1.添加新ItemType枚举

在 TouchFish.Scripts.Item.ItemType(/Assets/TouchFish/Scripts/Item/ItemType.cs) 中添加新的枚举Key

namespace TouchFish.Scripts.Item
{
    public enum ItemType
    {
        None,
        HealthPotionSmall,
        PoisonPotionSmall,
    }
}

2.创建Item元数据

在/Assets/TouchFish/Data/ItemData/目录下右键新建”Create/TouchFishGames/ItemData”，以”ItemData{物品名}”形式命名。

鼠标悬停在具体标签上可查看说明，按说明填充元数据即可。后续可以考虑扩展从csv一键生成元数据的功能。

3.创建Item可执行脚本

在/Assets/TouchFish/Scripts/Item/目录下右键新建”Create/TouchFishGames/Item Template”，以”Item{物品名}”形式命名。

修改type变量为该脚本对应的ItemType(不可为默认的ItemType.None)。然后实现对应函数即可。
