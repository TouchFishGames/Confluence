TouchFishGames : 背包系统设计

Created by Forrest Yang, last modified on 12月 23, 2022

基于目前的物品系统和配方系统，设计实现当前的生存模式风格的背包系统。

设计要点：

  • 有限的格子（可扩展）

  • 快速插入/删除，快速交换

  • 兼容重复key的情况（两个格子存同一种物品）

  • 拆分失败的处理

首先封装一个背包物品类：

BagItem {

ItemSO（考虑到渲染图标，以及是否能装备，是否能使用等的查询）

number 数量，

……

}

然后背包系统作为一个BagController 挂在Player身上

BagController 中核心变量为

BagItem[]

数组 ， 长度初始就确定，（生存游戏的背包不可能无限大小）

MaxCapacity

当前能用的背包容量

CurrentCapacity

当前已使用的容量

FirstPos

第一个空位置 （FirstPos >= MaxCapacity || CurrentCapacity >= MaxCapacity ）都表示背包满了

方法：

swap

交换物品（同物品触发堆叠）

add

添加物品

remove

物品

split

拆分
